feedback2 이후 추가 수정사항 (feedback3)

1) 리치 에디터/멘션
- 이미지 외 일반 파일 첨부 업로드를 지원하도록 확장
- 첨부파일 렌더링/다운로드 동작 보강
- 멘션 입력 시 '@이름' 형태로도 사용자 매칭되도록 보정

2) 시간/표시 기준
- 프론트엔드 시간 표시를 KST(Asia/Seoul) 기준으로 통일

3) 코칭 계획/실적 개선
- 기본 화면을 코치별 월 단위 관리 형태로 개선하고 기존 형태는 선택적으로 유지
- 계획 입력에서 과제 선택 제거
- 실적 입력에서 복수 과제 선택 지원
- 코치도 본인 실적 입력/수정 가능하도록 권한 확장
- 코칭 계획/실적 진입 시 이번 달이 기본 선택되도록 변경
- 날짜 필터를 추가하여 기본은 '코칭일정만' 노출, 필요 시 '전체' 전환 가능
- '코칭일정만' 선택 시 월 선택 없이 전체 코칭 일정 기준으로 표시
- '전체' 전환 시 불필요한 스크롤 점프가 발생하지 않도록 보정

4) 코치 소개/권한 노출
- 코치에게 할당된 과제가 코치 소개에 노출되도록 연계 보강
- 참관자(Observer) 권한 정책 보강: 쓰기 권한 없이 조회 전용
- 참관자가 권한 있는 과제 정보와 코칭일정을 정상 조회하도록 접근 제어 보완

5) 캘린더/코칭 연동
- 캘린더 공개 범위에 '코칭 일정'을 포함
- 코칭 일정과 코칭 계획/실적 데이터를 연동하여 상세에서 확인 가능하게 개선
- 공통 일정/코칭 일정 생성은 관리자만 가능하도록 정책 고정
- 캘린더에서 마일스톤(10주 마일스톤/Task 이벤트) 노출 및 편집 경로 제거

6) 대시보드 정합성
- 출석현황, 코칭노트/코칭의견 지표를 코칭 계획/실적과 동일한 날짜 필터 방식으로 통일
- '코칭일정만/전체' 필터 전환 시 합계/비율이 선택된 날짜 기준으로 재계산되도록 보정

7) 테스트/문서
- 캘린더 권한/노출 정책(참여자/참관자/관리자) 테스트 보강
- 마일스톤 비노출, 참여자의 공통/코칭 일정 생성 금지(403) 테스트 추가
- README 최신 변경사항 반영
