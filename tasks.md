  - 과제 목록 우측 상단 검색 버튼, 검색 화면 복귀 버튼 추가

- [x] 과제 목록 화면 전면 개편

  - 카드형/상태 태그(진행중/준비중/공개) 중심 표현 축소

  - 상단 히어로/운영정보/과제삭제 제거
  - 기본정보에 Github Repository(복수), 과제 요약, 진행률 조정 UI 추가
  - 기본정보 우측에 직전 코칭노트/코칭의견 위젯 추가

  - 문서 타입 UI를 아코디언(선택 시 펼침)으로 개편
  - 리치 에디터 작성 영역 최대화

- [x] 코칭노트 화면 재구성 1차

  - 작성/수정 화면: 템플릿/이력 제거, 날짜·주차 자동입력, 대형 에디터

### P1 (운영 기능 재정의)

- [x] 입실/퇴실 UX 개선

  - 공지 admin 전용/상단 고정/홈 주간 공지 노출
  - 목록 컬럼: 번호/분류/제목/작성자/날짜/조회수/댓글수
  - 본문·댓글 작성 모달 확대, 댓글 수정 지원
  - 게시글 이력/복원 기능 제거

- [x] 대시보드 지표 재정의

  - 과제별·날짜별 출석 인원/출석률
  - 과제별·날짜별 코칭노트 건수/코칭의견 작성 코치 수
