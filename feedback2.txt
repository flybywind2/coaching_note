1. 입실/퇴실 버튼이 안보임, 세션에서만 입실/퇴실이 적용되도록 코드설계 되어 있어서 그런거 같음
   세션 기능은 완전 삭제하고 세션에 상관없이 입실/퇴실이 날짜별로 기록되도록 수정

2 관리자 메뉴 사용자 관리
- 일괄 등록 기능에 이메일 입력은 필요없도록 수정 (이메일은 knox ID+@samsung.com으로 자동 생성)
- 삭제는 단순 비활성이 아닌 완전 삭제로 수정
- 사용자를 복수 선택해서 일괄 삭제하거나 일괄 수정하거나 있는 기능 추가
   (일괄 수정에서 knox ID, 이름, 이메일 수정은 비활성화하고
    부서, 역할, 차수 권한, 과제 권한 중 클릭해서 값을 입력한 항목만 일괄 수정 )
- 사용자 수정에서 차수 권한 -> 차수로 변경
- 차수 접근 권한 삭제, 모든 사용자는 모든 차수에 접근 가능
- 차수는 코치와 참여자가 어떤 차수에 해당되지는 구분하기 위한 것임
- 사용자 수정에서 차수와 과제 권한 체크 박스 너비 최소화하고 좌측 정렬

3 관리자 메뉴 차수 관리
- 과제 차수에 코칭 시작일 추가
- 주차는 코칭 시작일 기준으로 계산

4. SSP+소개 > 코치 소개
- 코치도 과제목록과 마찬가지로 차수별로 표시
- 코치 추가 버튼 삭제, 화면 하단 기본 코치 사용자 자동 표시 삭제
- 차수를 선택하면 해당 차수의 모든 코치가 자동 표시되며 관리자가 숨김/표시 결정
- 관리자가 코치별 사진, 코칭분야, 경력 입력하고 입력되지 않은 빈킨은 숨김
- 코치 카드는 2열로 배치하고 관리자가 코치 카드를 드래그해서 레이아웃을 수정할 수 있는 기능 추가

5 과제 목록
- 과제를 정식과제(primary)와 준참여과제(associate)로 구분
- 화면 상단에 본인 과제 리스트를 다른 과제들 리스트를 표시한 것과 같은 테이블형태로 표시
- 전체 과제리스트에서 본인 과제 리스트는 삭제
- 전체 과제리스트는 정식과제와 준참여과제를 탭으로 구분

6 과제 목록 > 기본 정보
- "기본 정보" "과제 운영 핵심 정보를 확인합니다" 문구 삭제, 이 위치에 과제명을 크게 표시
- GitHub repository를 과제 요약보다 상단에 배치
- 기본정보 편집 버튼 삭제하고 각 항목마다 작은 edit버튼을 추가
- 과제 팀원 추가시 부서명이 함께 표시되고 이름순이나 부서명 순으로 정렬 가능하도록 수정
- 과제 팀원 추가시 한명씩 여러번 추가하지 않고 사용자를 복수로 선택할 수 있게 수정
- 우측 코칭 노트에 최신 코칭 노트의 현재 상태만 표시하지 말고 최신 코칭 노트 전체 내용, 전체 코칭 의견과 메모가 모두 표시되도록 수정

8 과제 목록 > 과제 기록
- 세로 아코디언 구조를 좌측 고정 탭 구조 변경
- 기초컨설팅 -> 기초컨설팅 산출물
- 워크샵결과 -> 공동워크샵 산출물
- 중간 발표 -> 중간 발표 자료
- 최종 발표 -> 최종 발표 자료

9 과제 목록 > 코칭 노트
- 날짜 옆에 몇주차인지 표시
- 내용이 없는 항목은 빈칸으로 두지 말고 표시하지 않기
- 코칭 노트별로 접고 펼치기 가능하도록 수정, 최신 노트만 펼치고 나머지 노트는 기본으로 접기  (모두펼치기 버튼도 추가)
- 코칭 노트 작성 화면에 날짜, 주차에 자동 입력이란 문구 삭제
- 코칭 노트 작성 화면에 과제명 표시하고 과제명 우측에 진행률 입력 메뉴 배치 (너비는 작게)
- 과제 진행률 수정한 것이 기본 정보 과제 진행률에도 반영되도록 수정
- 코칭 노트 작성 화면에서 주요 문제 작성과 체크버튼을 수평으로 배치하고 체크 버튼 너비 최소화
- 코칭 노트 임시 저장 및 복원 기능 삭제

10. 코칭노트 버그 수정
- 코칭 노트 입력 후 기본 정보로 이동하면 새로 입력한 코칭 노트가 반영 안되어 있고 다시 코칭 노트로 돌아오면 새로 입력한 코칭 노트가 안 보임
- 화면 새로 고침을 해야 새로 작성한 코칭 노트가 보임
- 기본 정보의 우측 코칭 노트를 클릭하면 가장 최초의 코칭 노트 외에 다른 코칭 노트로는 링크 이동이 안됨


11. 캘린더
- 과제별 캘린더는 체크박스로 전부보여주는것이 아니라 과제를 선택해서 볼수 있게
- 일정 추가 화면에서 "일정 카테고리" 항목은 삭제하고 색상을 선택할수 있게 수정, 선택된 색상으로 캘린더에 표시
- 일정 추가 화면에서 과제 특수 일정에 "과제 일정 유형", "참석 코치" 항목은 삭제하고 "대상 과제" 선택 메뉴를 "제목" 보다 위로 배치
- 반복일정은 횟수 외에도 종료 날짜를 입력할수 있도록 수정 (해당 차수 종료일은 초과할 수 없음)
- 반복일정은 일괄 수정과 삭제도 가능하도록 기능 추가
- 일정 상세에 "유형"과 "코치배정" 삭제하고 없는 항목은 빈칸으로 두지 말고 보이지 않기
- 일정 상세에 "과제"는 "제목" 바로 아래 배치
- 일정 상세에 종일 일정이면 날짜 대신 종일 일정으로 표시
- 일정 수정 화면도 일정 추가 화면과 동일하게 변경
- 전체 일정은 하루 한개만 표시 가능
- 전체 일정에 상세에는 참여하는 코치 명단과 참여시간 표시 (12번 코칭 계획/실적과 연동)


12. 코칭 계획/실적
- 캘린더에 전체 일정이 입력된 날짜만 표시
- 주차별로 표시하지 말고 해당 차수의 시작일과 종료일을 모두 보여주고 스크롤바로 이동,
- 기본 스크롤 위치는 오늘날짜가 가장 왼쪽에 오도록 조정
- 계획이 입력된 날은 다른 색상으로 표시
- 코칭 계획 입력 화면에 과제 아래 "종일" 체크 박스를 추가하고 기본적으로 체크 상태 유지
- 코칭 계획 입력 화면에 "종일" 체크 박스를 해제하면 시작시간과 종료시간 입력화면 활성화
- 모든 날짜마다 계획과 실적이라는 글자를 적지 말고 코치이름 옆에 계획, 실적 두개의 행으로 구분하고 날짜 아래는 값만 표시
- 계획에 과제명은 표시안함
- 실적에 로그 건수 표시 안하고 시작시간 - 종료시간 (xx시간xx분) 형식으로 표기
- 계획이 입력된 코치는 캘린더 해당일의 전체 일정에 명단과 참여시간 표시 (11번 캘린더 기능과 연동)

13.게시판
- 공지사항은 관리자만 작성 가능하도록 수정
- 공지사항은 게시판 가장 위에 오도록 수정
- 검색 기능 추가

14. 대시보드
- 과제별 진행률이 가장 위에 오게 배치
- 출석 현황은 과제명을 세로축으로 두고 과제명 오른쪽 옆에 대상인원 표시
- 가로축을 날짜로해서 해당차수별 시작일과 종료일까지 캘린더에 전일 일정이 있는 날을 모두 표시하고 스크롤바로 이동
- 날짜마다 출석인원(출석률) 표시
- 코칭 노트/코칭 의견 지표도 가로축을 날짜로해서 해당차수별 시작일과 종료일까지 캘린더에 전일 일정이 있는 날을 모두 표시하고 스크롤바로 이동,
- 과제명을 세로 축으로 두고 과제 하나에 "코칭노트건수"와 "코칭 의견 작성 코치수" 두행으로 구분 날짜별 값을 표시, 가장 좌측에는 합계 표시


15. 리치 에디터
- 문단 형식에 본문 옵션이 적용되지 않는 버그 있음
- 리본버튼을 MS오피스와 같은 이름이나 아이콘으로 수정
- 표 삽입할때 PPT처럼 grid방식으로 추가하거나 Prompt Dialog로 두번 띄워서 가로, 세로값을 한번씩 받지 말고 모달창을 띄워서 한번에 받을 수 있게 수정
- 좌측, 중앙, 우측 정렬 버튼은 3개가 나란히 배치되도록


16. 공통 수정 사항
- 체크박스와 체크박스 제목은 수평으로 배치하고 체크박스의 너비는 최소화
